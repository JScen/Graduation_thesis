# 技能実習生の失踪予測 / 技能实习生失踪预测
### Predicting Disappearances of Foreign Technical Intern Trainees

機械学習を用いて、外国人技能実習生の失踪を予測する研究のコードリポジトリです。
本项目使用机器学习方法，预测外国人技能实习生的失踪情况。

---

## 📋 研究概要 / 研究概要

**日本語**
本研究は、技能実習生の選抜時点で取得可能な属性情報（年齢・性別・職種・所在地・派遣会社など）のみを用いて、その後の失踪を予測できるかを検証する。約3,000件のデータに対し、4つの分類モデル（決定木・ランダムフォレスト・ロジスティック回帰・SVM）を Leave-One-Out 交差検証（LOOCV）で評価し、失踪者（少数クラス）の検出性能を比較する。さらに、各モデルが見逃した失踪者（偽陰性）の特徴を分析する。

**中文**
本研究验证：仅使用技能实习生在选拔阶段可获取的属性信息（年龄、性别、职种、所在地、派遣公司等），能否预测其之后的失踪。基于约 3,000 条数据，用留一交叉验证（LOOCV）评估 4 个分类模型（决定树、随机森林、逻辑回归、SVM），比较其对失踪者（少数类）的检出性能。并进一步分析各模型漏判的失踪者（偽陰性 / False Negative）的特征。

### 主処理 / 主流程

| ファイル / 文件 | 説明（日本語） | 说明（中文） |
